# 🎉 LightDo v1.0.0 首个正式版发布

**一键下载，开箱即用！** 🚀

这是 LightDo 的首个正式版本，提供完整的待办事项管理功能和流畅的用户体验。

---

## ✨ 核心特性

- 🎯 **悬浮球交互** - 桌面右上角半透明悬浮球，点击即可快速访问待办列表
- ⚡ **开箱即用** - 解压即用，**无需 Node.js、npm 等开发工具**
- ✅ **完整待办功能** - 添加、编辑、删除、拖拽排序待办事项
- 🎨 **毛玻璃 UI** - Windows 原生 Acrylic 玻璃效果，现代简约设计
- ⌨️ **全局快捷键** - `Alt+Shift+T` 一键呼出/隐藏待办窗口
- 💾 **本地存储** - JSON 文件自动保存，隐私安全无云上传
- 🚀 **后台运行** - 支持开机自启，静默后台运行不打扰工作
- 🪟 **深度集成** - 完美适配 Windows 10/11，占用资源极少

---

## 🚀 极速开始（3 步搞定）

### 第 1 步：下载
- 点击下方 **Assets** 下载 `LightDo-v1.0.0-portable.zip` (103.97 MB)

### 第 2 步：解压
- 用 Windows 资源管理器解压文件到任意文件夹

### 第 3 步：运行
- 双击 `LightDo.exe` 立即使用
- **无需安装，无需配置！**

## 💻 系统要求

- **操作系统**：Windows 10 或更高版本
- **内存**：最低 256 MB
- **磁盘空间**：最少 200 MB 
- **网络**：无需联网
- **依赖**：完全独立，无任何额外依赖

---

## 📖 使用指南

| 操作 | 说明 |
|------|------|
| **点击悬浮球** | 显示/隐藏待办列表 |
| **Alt+Shift+T** | 全局快捷键，快速显示/隐藏 |
| **双击任务文本** | 编辑任务内容 |
| **拖拽任务** | 重新排序待办列表 |
| **勾选任务** | 标记任务完成/未完成 |
| **右上角齿轮** | 打开设置面板 |
